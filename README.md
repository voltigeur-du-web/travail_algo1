# Travail Algo 1
## Contenu du repo
* **advanced_dijkstra_algo** inclut une première version de l'algorithme utilisant des structures plus complexe à devoir démontrer. Néanmoins, cette version est celle que j'ai utilisé pour faire des appels depuis avec python.
* **backend** inclut le code python pour obtenir les données géographiques open street map et un wrapper pour convertir les structures de données de python en données utilisables par l'algorithme C.
* **report_algo_simplifié** est l'endroit où se situe le travail du cours d'algorithmique et son script en latex.
* **simplified_dijkstra_algo** inclut le code C décrit dans le travail du cours d'algorithmique.
* **Makefile** est un fichier incluant des commandes bash utiles pour préparer son environement de dévelopement ainsi que des commandes pour compiler et tester la première version de l'algorithme (advanced_dijkstra_algo) non utilisé dans le travail.
